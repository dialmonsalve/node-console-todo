# TODO APP CONSOLE

## This project is a TODO made in typescript and node.js, this is a console app

1. To install:

```shell
  npm init
```

2. If you don't have install typescript globally:

  ```shell
    npm i typescript
  ```

3. Create a script in the package json:

```json
  "scripts": {
    "tsc": "tsc"
  },
```

4. Transpile typescript to javascript:

```shell
  npm run tsc
```

5. Init app:

```shell
  node dist/app.js
```
